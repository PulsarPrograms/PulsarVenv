PulsarVenv 0.0.1-alpha - Справочник по командам (Для продолжения работы с пульсаром закройте этот файл)

Информация о PulsarVenv:
====================
Версия:            PulsarVenv 0.0.1-alpha
Платформы:         Windows
Доступные сборки:  0000 (полная), другие (ограниченные)

Основные команды:
=================
help               - Показать эту справку
exit               - Выйти из системы
clear              - Очистить экран консоли
pinfo              - Показать информацию о системе (версия, время работы и т.д.)

Команды работы с системой:
=========================
sysconfig          - Показать конфигурацию системы
sumulator_pulsar   - Запустить эмуляцию пульсара (требует подтверждения)

Команды для вычислений:
=======================
calc <выражение>   - Выполнить математическое вычисление (например: calc 2+2*3)

Работа с Python:
===============
python <скрипт>    - Запустить Python скрипт (доступно только в сборке 0000)

Управление аккаунтами:
=====================
account add        - Создать новый аккаунт
account remove     - Удалить существующий аккаунт
account swap       - Сменить текущий аккаунт

Запуск программ:
===============
<имя_программы>    - Запустить программу из папки modules
                    (можно указывать с расширением .exe или без)
